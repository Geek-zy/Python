
安装 Django
tra xvzf Django.x.y.tar.gz
sudo python3 setup.py install

查看 Django 命令
django-admin

创建项目
django-admin startproject myblog

Django 子命令
python3 manage.py

启动服务
python3 manage.py runserver

切换端口
python3 manage.py runserver 9999

创建应用
python3 manage.py startapp blog

创建模板
Django 中模板目录名必须是 templates
再在 templates 下新建一个以应用名称相同的目录
最后将 html 文件放置在应用名称的目录下

DTL Django 的内置模板
{{参数名}}

模板 for 循环
{% for xx in xxs %}
	HTML 语句
{% endfor%}

ORM 框架 => 对象关系映射

生成数据表 => 准备
python3 manage.py makemigrations

生成数据表 => 迁移
python3 manage.py migrate

查看 SQL 语句
python3 manage.py sqlmigrate blog 0001

配置 Admin
创建超级用户
python3 manage.py createsuperuser

Admin 设置中文
在settings.py 中将 LANGUAGE_CODE = 'en-us'修改为：LANGUAGE_CODE = 'zh_Hans'

将模型中的数据表放入到 Admin 后台
admin.py 文件里面配置

修改 Admin 显示的标题
models 加入 def __str__(self):

Django Shell
启动 Shell
python3 manage.py shell
